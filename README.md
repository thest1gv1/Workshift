# Workshift

Мобильное PWA приложение для сотрудников банка. Используется в течение рабочей смены для учёта клиентов и формирования отчёта в конце дня.

## Стек

- Next.js 16 + TypeScript
- Tailwind CSS v4 + CSS Modules
- shadcn/ui
- PostgreSQL + pg
- PM2 + Nginx (VPS)

## Что сделано

- Создал Next.js проект с TypeScript, Tailwind, shadcn
- Настроил дизайн систему — токены цветов, шрифты Onest и Unbounded
- Сверстал главный экран — карточка смены, список клиентов, кнопки
- Сверстал форму добавления клиента — поля, чипы услуг, тип
- Сверстал экран статистики и настроек
- Вынес компоненты — ShiftCard, StatBox, ClientRow, BottomNav
- Подключил Nano Stores — settingsStore и shiftStore
- Сделал рабочую логику — добавление клиентов, динамические цифры, прогресс план
- Задеплоил на Vercel

- Подключил PostgreSQL — создал БД, таблицу клиентов, настроил подключение через `pg`
- Написал API routes — GET, POST, PUT, DELETE для клиентов
- Реализовал редактирование клиента — загрузка данных по id, обновление через PUT
- Задеплоил на VPS с PM2 и Nginx — обратный прокси, несколько проектов на одном сервере
- Настроил PWA — манифест, иконки, поддержка установки на домашний экран
